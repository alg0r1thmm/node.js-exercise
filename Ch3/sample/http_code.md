# http 상태 코드

* 100번대 : 정보 / 서버가 요청을 받았고 클라이언트의 작업 진행을 허가
* 200번대 : 성공 / 요청을 성공적으로 받고 수용했다는 코드
* 300번대 : 리다이렉션 / 요청을 완료하기 위해서 추가적인 작업이 필요하다는 의미
* 400번대 : 클라이언트 서버 오류를 의미
* 500번대 : 서버 오류를 의미

<br/>

# 주요 사용 코드

* 200 : 요청 성공
* 204 : 요청은 성공했으나 제공할 내용 없음
* 304 : 이전 요청과 동일
* 400 : 클라이언트 요청 오류
* 401 : 요청을 위한 권한 요구
* 403 : 요청이 서버에 의해 거부
* 404 : 요청한 URL 찾기 불가
* 500 : 서버 오류 발생에 의한 응답 불가
